## Maintenance notes:

##### Merge process

Update process:

1. Fork the hed-standard/hed-resources repository in your own github account.
1. Clone this repository locally and create a new branch.
1. Make your changes and then push to that new branch on your own repository.
1. When you are done, do a pull request for hed-resources to the actual repository on hed-standard.
1. Ask for reviews.
1. When all reviewers have approved, the administrator will merge to the main branch of hed-standard.
